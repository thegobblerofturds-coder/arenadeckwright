PROFILE VAULT
- Named profiles store the deck name, exact gradient, locks, formatting, WUBRG selection and interface mode
- Profiles persist in local browser storage between sessions
- There is no artificial profile-slot limit
- Export All writes a portable JSON backup
- Import Backup restores or merges profiles on another browser or computer
- Profile controls load, update, duplicate, rename and delete

Nothing is uploaded. Local browser storage can be cleared by the browser or device, so export a JSON backup for long-term preservation.
